# react-sandbox

My sandbox for React

## Setup

```
$ git clone git@github.com:tayutaedomo/react-sandbox.git
$ cd react-sandbox
```

### without Docker

```
$ yarn install
$ yarn start
```

### with Docker

```
$ touch .bash_history
$ docker-compose build
$ docker-compose run --rm app yarn install
$ docker-compose up
```

## Available Scripts

```
$ yarn start
$ yarn test
$ yarn build
$ yarn eject
```
